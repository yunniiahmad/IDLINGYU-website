# IDLINGYU Website

Public marketing site for IDLINGYU (English for Kids &amp; General English tutoring),
deployed via GitHub Pages.

Source of truth for curriculum content lives in the private
[yunniiahmad/IDLINGYU](https://github.com/yunniiahmad/IDLINGYU) repo, under `docs/`.
This repo mirrors that folder so the site can be hosted publicly without exposing
the rest of the curriculum repo.

To update the live site: edit `docs/` in the main IDLINGYU repo, copy the changed
files here, commit, and push.
